EZ Parent
===
### EZ 构建框架，包含了Scala项目的基础依赖及构建插件。

##使用

    <dependency>
        <groupId>com.ecfront</groupId>
        <artifactId>ez-parent</artifactId>
        <version>0.1</version>
    </dependency>

### Check out sources
`git clone https://github.com/gudaoxuri/ez-parent.git`
